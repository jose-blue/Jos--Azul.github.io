Introducción a HTML

¿Qué es HTML?

HTML o (Lenguaje de Marcas de Hipertexto “HyperText Markup Language”), es un lenguaje que se utiliza para dar estructura y estilo a una página web o documento. Este no es un lenguaje de programación, en realidad es un lenguaje de marcado que está estructurado por etiquetas. (Raffino, 2023)

¿Para qué sirve HTML?

HTML nos ayuda para describir la estructura elemental de una página. Este lenguaje de marcado ordena el contenido de forma tal manera que muestre cómo este se va a visualizar. Nos ayuda a poder mostrar textos, imágenes, videos, etc. (Raffino, 2023)

¿Cómo funciona HTML?

El lenguaje de marcado o HTML funciona precisamente con marcadores escritos, los cuales se muestran entre comillas angulares: <html>. Con estos comandos puedes construir fácilmente tu web y añadir toda la información importante para todos los navegadores. También los tags para la estructura básica, además podemos incluir contenido multimedia, crear formularios o determinar el tipo de letra. (Raffino, 2023)

TAGS DE HTML PARA ORDENAR UNA WEB

  • <!DOCTYPE>: Nos ayuda a determinar el tipo de documento.
  • <html>: Sirve para que se defina como documento HTML.
  • <head>: Ayuda a almacenar los metadatos del documento.
  • <title>: Este comando contiene el título del documento, que además se mostrará en la barra de título del buscador.
  • <body>: Es la sección principal y engloba todo el contenido que verás en el navegador.
  • <nav>: Ayuda a la sección de navegación de una web.
  • <section>: Con esta etiqueta puedes resumir los elementos en un grupo.
  • <article>: Sirve para la sección de contenido de una web.
  • <header>: Esta etiqueta define el encabezado de una web o una sección.
  • <footer>: Esta etiqueta define el pie de página de una web o una sección.

TAGS DE HTML PARA ESTRUCTURAR UN DOCUMENTO

  • <h1> a <h6>: Los titulares van en los distintos tags h. Cuanto menor sea la cifra, mayor será el encabezado.
  • <p>: Nos indica que se trata de un párrafo.
  • <br>: Esta etiqueta fuerza un salto de línea.
  • <hr>: Crea una línea divisoria física entre dos tipos de contenido.

TAGS HTML PARA AJUSTAR LA TIPOGRAFIA

  • <b>: Sirve para poner en negrita palabras sueltas, frases o párrafos.
  • <i>: Sirve para poner en cursiva palabras sueltas, frases o párrafos.
  • <u>: Con esta etiqueta el fragmento seleccionado estará subrayado.
  • <s>: Con esta etiqueta el texto seleccionado estará tachado.
  • <sup>: Sirve para que el carácter seleccionado sea un superíndice, p. ej. 1o.
  • <sub>: Con esta etiqueta puedes escribir caracteres en subíndices, como el 2 de H2O.

TAGS PARA RESALTAR Y MARCAR PÁRRAFOS HTML

  • <strong>: Esta etiqueta sirve para resaltar párrafos.
  • <em>: Con esta etiqueta puedes enfatizar fragmentos de texto.
  • <q>: Esta etiqueta sirve para marcar citas dentro de un texto.
  • <blockquote>: Sirve para marcar todo un párrafo como cita.

TAGS DE TABLAS Y LISTAS HTML

  • <table>: Sirve para crear una tabla.
  • <caption>: Esta etiqueta para definir el título de la tabla.
  • <tr>: Esta etiqueta sirve para las líneas de la tabla.
  • <td>: Sirve para definir una casilla de la tabla.
  • <ul>: Esta etiqueta te ayuda crear una lista de viñetas.

TAGS DE CONTENIDO MULTIMEDIA HTML

  • <img>: Sirve para indicar una imagen.
  • <map>: Con esta etiqueta puedes insertar un mapa en un documento.
  • <audio>: Sirve para insertar un sonido.
  • <video>: Nos sirve para añadir contenido en vídeo.

OTROS COMANDOS IMPORTANTES DE HTML

  • <style>: Esta etiqueta introduce los códigos CSS que determina el aspecto de tu web.
  • <div>: Esta etiqueta sirve para definir ciertos párrafos de un documento.
  • <label>: Esta etiqueta se utiliza junto con <input> y define un campo de texto de <input>.
  • <iframe>: Sirve para incluir contenido externo en tu web.
